# 🧠 Probable

**Probable** is a Web3 prediction market platform that enables users to create and trade on the probability of real-world events using blockchain-based markets.

This project is being built as a **portfolio-grade decentralized application**, focusing on clean architecture, smart contract design, and real-world Web3 mechanics.

---

## 🚀 Overview

Prediction markets allow users to speculate on future events by buying shares in outcomes (e.g., YES/NO). Prices reflect the **probability of an event occurring**, driven by market activity.

**Probable** implements this concept on-chain using smart contracts.

---

## ⚙️ Features (Planned)

* Create prediction markets
* Buy YES / NO outcome shares
* Dynamic probability based on market participation
* Market resolution by oracle/admin
* Claim rewards for correct predictions
